GSM-Plug: control through GSM network.

   

How does it work  :


GSM-plug is an independent controller which can be guided through a GSM network avilable.

It is based on gsm-shield for arduino which accepts short messages (sms).

A simple communication protocol with simple commands is built in the GSM-plug in order to control any device conncted to the GSM-plug.

This is shown here with a lamp conncted to it that can be turned on or off through any mobile device by sending the relevant command.
